# Um cabeçalho H1 #

Parágrafos são separados por uma linha em branco.

Segundo parágrafo. *Itálico*, **negrito**, and `monoespaçado`. Listas de items
ficam assim:

 * item
 + outro item
 - mais outro item

Note que --- não considerando os sinais --- o conteúdo de texto começa à
partir da quarta coluna. Você também pode usar outros sinais para
representar a lista.

> Blocos de citação
> se parecem com isso.
>
> Eles podem ter múltiplos parágrafos,
> se você quiser.

Use três traços para travessão. Use dois traços para intervalos (ex., "está
tudo entre os capítulos 12--14"). Três pontos ... serão convertidos.
Unicode é aceito também ☺!

## Um cabeçalho H2 (com ID) ## {#h2-com-id}

Aqui uma lista numerada:

 1. item 1
 2. item 2
 3. item 3

Here's a code sample:

    # Let me re-iterate ...
    for i in 1 .. 10 { do-something(i) }

Como você pode ter adivinhado, a identação serve para criar um bloco de código.
Contudo, em vez de identar sempre o bloco, você pode usar delimitadores,
se você quiser:

~~~ c
void hello() {
    println("hello world!\0");
}
~~~

(o que torna colar o código mais fácil). Você pode opcionalmente usar o
liquid para adicionar realce de sintaxe:

{% highlight python %}
import time
# Rápido, contando até dez!
for i in range(10):
    # (mas não *tão* rápido)
    time.sleep(0.5)
    print i
{% endhighlight %}

{% highlight java %}
package br.com.tocadoelfo.ftweb.util;

import java.io.Serializable;
import java.util.List;
import javax.persistence.*;
import javax.persistence.criteria.*;

public class DaoGenericoJpaImpl <T extends Serializable> implements DaoGenerico<T> {
    
    protected Class<T> entidade;

    protected EntityManager em;
    
    protected DaoGenericoJpaImpl() { }
    
    public DaoGenericoJpaImpl(Class<T> entidade, EntityManager em) {
        this.entidade = entidade;
        this.em = em;
    }
    
    @Override
    public CriteriaBuilder getCriteriaBuilder() {
        CriteriaBuilder cb = em.getCriteriaBuilder();
        return cb;
    }
    
    @Override
    public CriteriaQuery<T> getEntidadesByProp(String prop, Object valor) {
        CriteriaBuilder builder = this.em.getCriteriaBuilder();
        CriteriaQuery<T> cqt = builder.createQuery(this.getEntidade());
        
        Root<T> re = cqt.from(this.getEntidade());
        Predicate eq = builder.equal(re.get(prop), valor);
        cqt.select(re).where(eq);
        
        return cqt;
    }
}
